Database upgrade to v220 featuring AI Easyscan, enhanced MyInvois automation, expanded Mobile Connect features, restored online document approval, and new reporting capabilities. (*890*)

<!-- truncate -->

## Updates

- **MyInvois (E-Invoice MY)**:
  - Exclude EIVRequest_UUID when transfer document
- **Ecommerce**:
  - Adjustment to Shopee return quantity calculation
- **Approval**:
  - Once a document has been approved, re-editing an document shouldn't allow change to **Draft** or **Draft Rejected**
  - Acceptable transaction date rules require re-approval whenever user edit document again
- **InvoiceNow (SG)**:
  - Exclude `Peppol_UUID` and `Peppol_DocUUID` when transfer document

## Bug Fixes

- Fixed a bug where E-Invoice Request ignoring E-Invoice start date
- Job Order, Stock Assembly & Disassembly should prompt document validation dialog if transfer from other document
