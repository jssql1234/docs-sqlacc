Sales/Purchase UI improvements, currency rate refresh for Cash Book & Journal Entry, and various bug fixes. (*894*)

## Updates

- **InvoiceNow (Peppol SG)**:
  - Added GST validation check before activating GST InvoiceNow.

- **Cash Book / Journal Entry**:
  - Added currency rate refresh button to Cash Book and Journal Entry (#7873).

## Bug Fixes

- **Sales / Purchase**:
  - Fixed next number label alignment in Sales/Purchase Documents to prevent overlapping (#7875).
  - Fixed Customer/Supplier Payment UI layout (#7834).

- **General**:
  - Fixed year-end memory leak.
  - Fixed document form overlapping after using Tile Vertical (#7827).
  - Fixed SQL Control Center prompting access violation on Windows 64-bit (#7872).

- **E-Commerce**:
  - Excluded Shopee shipping charges from partial refund calculation.
