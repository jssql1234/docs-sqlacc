## Steps and Instructions

1. You must make sure setting had turn on the subitem setting

    ![1](../../../static/img/integration/pos/sub-item/sub-item-setting.png)

2. Press OK and you will see Sub Item on home screen

    ![2](../../../static/img/integration/pos/sub-item/sub-item-setting1.png)

3. Press New and create the sub item accordingly

    ![3](../../../static/img/integration/pos/sub-item/sub-item-setting2.png)

4. After creating the subitem, click on category and generate new code to add in sub item template under category

    ![4](../../../static/img/integration/pos/sub-item/sub-item-setting3.png)

    ![5](../../../static/img/integration/pos/sub-item/sub-item-setting4.png)
